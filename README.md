# Kanban MVP Frontend

Single-board Kanban web app built with Next.js (App Router, TypeScript, client-side state).

## Run

- `npm run dev`

## Quality

- `npm run lint`
- `npm run test`
- `npm run test:e2e`

## Scope

- Fixed 5 columns (renameable)
- Card title/details only
- Drag and drop card movement
- Add/delete cards
- Dummy seeded data
- No persistence, auth, search, or archive
